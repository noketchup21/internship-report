#### Create input file

1. On your computer, create a file named ``test.txt``. 
+ Create with the text: "Hello, congratulations on completing the workshop!"
![test](/images/5.5-testing/create-file.jpg)
+ Save file
1. Return to the **S3 tab**, open the **input** folder.
![input](/images/5.5-testing/input.jpg)

1. Click **Upload** -> Select ``test.txt`` -> Click **Upload**
![upload](/images/5.5-testing/upload.jpg)
1. Wait for about 5 seconds.
   
2. Go back to the Bucket console, then open the output folder.
![output](/images/5.5-testing/output.jpg)
1. You will see the file test.mp3. Select it and click Download.
2. Open it to listen.
