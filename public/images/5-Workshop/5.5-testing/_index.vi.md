#### Tạo file đầu vào

1. Trên máy tính của bạn, tạo một file có tên là ``test.txt``. 
+ Tạo file với nội dung: "Hello, congratulations on completing the workshop!"
![test](/images/5-Workshop/5.5-Policy/create-file.jpg)
+ Lưu file lại.
2. Quay lại **tab S3**, mở thư mục **input**.
![input](/images/5-Workshop/5.5-Policy/input.jpg)

3. Nhấn **Upload** -> Chọn file ``test.txt`` -> Nhấn **Upload**
![upload](/images/5-Workshop/5.5-Policy/upload.jpg)
4. Chờ khoảng 5 giây.
 
5. Quay lại giao diện Bucket, sau đó mở thư mục **output**.
![output](/images/5-Workshop/5.5-Policy/output.jpg)
6. Bạn sẽ thấy file test.mp3. Chọn nó và nhấn **Download**.
7. Mở file lên để nghe thử.
