#### Create input file

1. On your computer, create a file named ``test.txt``. 
+ Create with the text: "Hello, congratulations on completing the workshop!"
![test](/images/5-Workshop/5.5-Policy/create-file.jpg)
+ Save file
2. Return to the **S3 tab**, open the **input** folder.
![input](/images/5-Workshop/5.5-Policy/input.jpg)

3. Click **Upload** -> Select ``test.txt`` -> Click **Upload**
![upload](/images/5-Workshop/5.5-Policy/upload.jpg)
4. Wait for about 5 seconds.
   
5. Go back to the Bucket console, then open the output folder.
![output](/images/5-Workshop/5.5-Policy/output.jpg)
6. You will see the file test.mp3. Select it and click Download.
7. Open it to listen.
